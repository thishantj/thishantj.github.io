i-FinePay

Project information

  • Category: Mobile development
  • Project: Research project
  • Publication URL (DOI): 10.5120/ijca2021921720

The project was designed to be a research project which was published in the IJCA journal. Project topic was set as "Platform Independent on the Spot Traffic Payment Solution," designed to work in the law enforcement of Sri Lanka. The project included a mobile application, web application and image processing.
