Proximity Bus stop locator

Project information

  • Category: Design
  • Client: Part of course module
  • Project URL: Figma design

The design task was a prototype design of a application designed to get the location of bus stops within the proximity of a selected playground in Helsinki. The application was designed and prototyped in Figma. The design was made for the purpose of running as a desktop application. The application contained features such as favouritising playgrounds according to the user's selection and searching for playgrounds within a given search radius.
